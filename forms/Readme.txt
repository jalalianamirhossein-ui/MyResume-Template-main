===============================================
CONTACT FORM - README
===============================================

This directory contains the contact form handler for the Meet AJ Portfolio.

Files:
- contact.php: Secure contact form processor with validation
- Readme.txt: This file

Features:
- Input validation and sanitization
- Rate limiting protection
- XSS and injection prevention
- Email format validation
- Secure email processing

Note: The contact form is fully functional and includes
security measures to prevent spam and malicious submissions.

For support or customization, visit: https://meetaj.ir
